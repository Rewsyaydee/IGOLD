# Kaifiat (Prayer Posture) Videos

Place kaifiat (solat posture) video files here for the **default** instructional model.

## Required files

| Filename | Posture | Format | Specs |
|----------|---------|--------|-------|
| `stand.mp4` | Standing (Qiyam) | MP4 | 9:16 portrait, muted, looping, ~3–6 s |
| `takbir.mp4` | Takbiratul Ihram | MP4 | 9:16 portrait, muted, looping, ~3–6 s |
| `qiyam.mp4` | Standing & Reciting | MP4 | 9:16 portrait, muted, looping, ~3–6 s |
| `ruku.mp4` | Bowing (Ruku') | MP4 | 9:16 portrait, muted, looping, ~3–6 s |
| `itidal.mp4` | Rising (I'tidal) | MP4 | 9:16 portrait, muted, looping, ~3–6 s |
| `sujud.mp4` | Prostration (Sujud) | MP4 | 9:16 portrait, muted, looping, ~3–6 s |
| `duduk.mp4` | Sitting Between Sujud | MP4 | 9:16 portrait, muted, looping, ~3–6 s |
| `tashahhud.mp4` | Tashahhud Sitting | MP4 | 9:16 portrait, muted, looping, ~3–6 s |
| `salam.mp4` | Closing Salam | MP4 | 9:16 portrait, muted, looping, ~3–6 s |

## Notes

- Videos should be **muted** (no audio track). Audio is handled separately.
- Keep file sizes small: target < 2 MB per file at 720p or lower.
- Use **portrait orientation (9:16)** for mobile-first display.
- Short looping clips (3–6 seconds) work best.
- Videos are displayed in the Kaifiat step-by-step and Posture Explorer sections.
