# Janazah Audio Files

Place janazah (funeral prayer) recitation audio here.

## Required files

| Filename | Step | Format |
|----------|------|--------|
| `niyyah.mp3` | Intention for Janazah Prayer | MP3 |
| `takbir-1-fatihah.mp3` | First Takbir — Al-Fatihah | MP3 |
| `takbir-2-selawat.mp3` | Second Takbir — Salawat | MP3 |
| `takbir-3-doa.mp3` | Third Takbir — Supplication | MP3 |
| `takbir-4-salam.mp3` | Fourth Takbir — Salam | MP3 |

## Notes

- MP3 format, 128 kbps or higher.
- Clear recitation of the Arabic text.
- If a file is missing, a placeholder chime plays instead.
